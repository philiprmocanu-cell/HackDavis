# RTK Curriculum Code Directory

> **Purpose:** A master directory of "RTK codes" that map to individual self-study chapters designed to fill the educational gaps documented in `haryana-education-context.md`. A user texts a code (e.g., `RTK29898`) and receives the corresponding chapter via SMS.
>
> **Design principle:** Each chapter takes the topic that Haryana's school system covered weakly, skipped, or skipped operationally — and re-teaches it using American K–12 / community-college pedagogy: stepwise, application-anchored, plain-language, with worked examples in the local Indian / Haryanvi context (₹, kirana, mandi, PHC, Aadhaar, UPI, Haryana districts). When the original Haryana subject was reasonable, the chapter mirrors it but adds the practical, civic, and digital extensions American curricula treat as core.
>
> **Code format:** `RTK` + 5-digit numeric code. The first digit identifies the subject family. Codes are stable; new chapters are appended, never re-used.
>
> **Subject families:**
> - `RTK1xxxx` — Languages & Literacy
> - `RTK2xxxx` — Mathematics & Numeracy
> - `RTK3xxxx` — Science (Life, Physical, Earth, Environment)
> - `RTK4xxxx` — Digital & Financial Literacy
> - `RTK5xxxx` — Civic, Legal & Government Procedures
> - `RTK6xxxx` — Health, Body & Mental Wellness
> - `RTK7xxxx` — Career, Livelihood & Entrepreneurship
> - `RTK8xxxx` — Agriculture & Rural Economy
> - `RTK9xxxx` — Critical Thinking, Media Literacy & Soft Skills
>
> **Entry format (one line per chapter):**
> `RTKxxxxx — <Title> — <One-sentence description of what the chapter teaches and the gap it fills.>`
>
> **Authoring source:** Chapters can be (a) excerpts/adaptations from open-licensed American textbooks (CK-12, OpenStax, Khan Academy text, OpenSciEd, EngageNY, Illustrative Mathematics) translated/localized to Haryana, or (b) AI-generated chapters following the delivery guide. The directory does not specify the source — it specifies what the chapter must teach. The delivery guide governs how it is sent.
>
> **Length:** Chapters are deliberately long when the topic demands depth (e.g., compound interest, FIR filing, antenatal care, scam recognition); short and tight when the topic is narrow (e.g., a single math drill). The delivery guide governs SMS chunking.

### Conventions

- A user texts the code (e.g., `RTK29898`) to the gateway number.
- The system finds the chapter, segments it per the delivery guide, and replies in the user's preferred language.
- Multiple codes can be queued (`RTK43008 RTK48002`).
- Modifiers (e.g., `Hindi RTK43008`, `Slow RTK43008`, `Short RTK43008`) are handled per the delivery guide.
